Note

In working through this project, I ran into issues with glue and Plotly figures. Doing some research, I found a function that overcame this issue. Here’s the function:

import panel as pn
pn.extension('plotly')

def gluePlotly(name,fig,**kwargs):
    """Wrap Plotly object with Panel and glue()"""
    return glue(name, pn.pane.Plotly(fig,**kwargs), display=False)
